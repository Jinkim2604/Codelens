# Frontend

This folder contains the Next.js frontend for CodeLens.

For full project setup, environment variables, local development, and API details, see the root README:

[README.md](c:\Users\jinki\OneDrive\Bilder\Desktop\CODElens\codelens\README.md)

## Local Commands

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

The frontend expects the backend to be available at `http://localhost:3001`.
